Como proposto, fiz um pequeno chatbot simulando uma pizzaria delivery.
Inicialmente há um cardápio de pizzas para a escolha do cliente,
após, um cardápio de bebidas,
e para a localização do cliente, usei a funcionalidade de enviar a localização para o bot
que, por sua vez recebe as coordenadas geográficas fornecidas pelo GPS.
Usando a API Geocoding, do Google Maps, faz-se uma requisição HTTP com as coordenadas, para o servidor Google,
é retornado um JSON com o endereço completo da localização.
É impresso na tela o pedido e o endereço do cliente.
